How to use tags in HTML

In HTML there are specific tags that the language uses in order to

seperate

text, paragraphs, headers, bold text, and italizing text. All the fluff will be inside of the <'body'> and <'p'> with <'/p'> creating paragraph beginning and end points (the quote marks ' ' are omitted in practice).

For making a header you use <'h1'> and <'/h1'> and the text to be bold in the middle (minus the quote ' ' marks)

To italize text you need to surround it with <'em'> and <'/em'> (minus the quote ' ' marks as well).

In order to make comments we will need to use <'!'-- and '--'> (no ' ' in practice) to create the comment space. Any text that is inside the comment space will not be seen on the web page.
